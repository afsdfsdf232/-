Web Vitals

Google 在 2020年5月5日提出了新的用户体验量化方式 Web Vitals 来衡量网站的用户体验，并将这些衡量结果作用于其排名算法的一部分。

Core Web Vitals 是应用于所有web页面的 Web Vitals 的子集，是其最重要的核心

核心指标：

  • 加载性能（LCP） 显示最大内容元素所需的时间
  • 交互性（FID）首次输入延迟的时间
  • 视觉稳定性（CLS）累计布局配置偏移


测量 Web Vitals

  • 性能测试工具，比如 Lighthouse
  • 使用web-vitals 库
  • 使用浏览器插件 web Vitals
